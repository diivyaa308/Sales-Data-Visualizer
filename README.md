# Sales-Data-Visualizer
Focuses on analyzing sales data using Python and visualizing insights using Power BI. The dataset was processed using Pandas, and visualizations were created using Matplotlib to calculate total sales, profit, and monthly trends. Charts like bar, line, pie, and KPI cards help understand product performance, regional distribution, and sales growth.

## Tools Used
- Python
- Pandas
- Matplotlib
- Power BI

## Features
- Sales analysis
- Monthly trend visualization
- Regional sales distribution
- Interactive dashboard
